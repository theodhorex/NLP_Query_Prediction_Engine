Pembelajaran mesin adalah subbidang dari kecerdasan buatan yang memungkinkan sistem untuk belajar dari data tanpa diprogram secara eksplisit. Algoritma pembelajaran mesin membangun model matematis berdasarkan data pelatihan untuk membuat prediksi atau keputusan. Pembelajaran mesin digunakan dalam berbagai aplikasi seperti filter spam, rekomendasi produk, dan pengenalan wajah. Terdapat tiga jenis utama pembelajaran mesin yaitu supervised learning, unsupervised learning, dan reinforcement learning. Supervised learning menggunakan data berlabel untuk melatih model sedangkan unsupervised learning menggunakan data tanpa label. Reinforcement learning menggunakan sistem reward untuk melatih agen membuat keputusan.
